## HTML Introduction

## HTML 소개

### What is HTML?

### HTML이란 무엇인가?

HTML is a markup language for describing web documents (web pages).

HTML은 웹 문서(웹 페이지)를 설명하기 위한 마크업 언어이다.

* HTML stands for Hyper Text Markup Language
* A markup language is a set of markup tags
* HTML documents are described by HTML tags
* Each HTML tag describes different document content

* HTML은 Hyper Text Markup Language의 약자이다.
* markup언어는 markup태그 세트이다.
* HTML 문서들은 HTML태그에 의해 설명됩니다.
* 각각의 HTML태그는 다른문서의 내용을 설명합니다.

### HTML Example

### HTML예

###  A small HTML document:

### 짧은 HTML문서:

<pre class="prettyprint">
&lt;!DOCTYPE html&gt;
&lt;html&gt;
&lt;head&gt;
&lt;title&gt;Page Title&lt;/title&gt;
&lt;/head&gt;
&lt;body&gt;

&lt;h1&gt;My First Heading&lt;/h1&gt;
&lt;p&gt;My first paragraph.&lt;/p&gt;

&lt;/body&gt;
&lt;/html&gt;
</pre>

---

### Example Explained

### 설명 예

* The DOCTYPE declaration defines the document type to be HTML
* The text between <html> and </html> describes an HTML document
* The text between <head> and </head> provides information about the document
* The text between <<title> and </title> provides a title for the document
* The text between <body> and </body> describes the visible page content
* The text between <h1> and </h1> describes a heading
* The text between <p> and </p> describes a paragraph

* DOCTYPE선언은 HTML문서가 될수 있는 정의 이다.
* 문자 사이에 <html>그리고</html> HTML문서가 그려진다.
* 문자 사이에 <head>그리고</head> 문서의 정보를 거의 제공한다.
* 문자 사이에 <<title>그리고</title> 문서의 제목을 제공한다.
* 문자 사이에 <body>그리고</body> 보여지는 페이지의 내용을 설명한다.
* 문자 사이에 <h1>그리고</h1> 제목을 설명한다.
* 문자 사이에 <p>그리고</p> 단락을 설명한다.

Using this description, a web browser can display a document with a heading and a paragraph.

이것은 일부 사용, 웹 브라우저 제목과 단락을 함께 문서를 표시할 수 있다.

---

### HTML Tags

### HTML 태그

HTML tags are keywords (tag names) surrounded by angle brackets:

HTML 태그는 키워드(태그이름)는 괄호안에 둘러싸여:

<<tagname>content</tagname>

<태그이름> 내용 </태그이름>

* HTML tags normally come in pairs like <p> and </p>
* The first tag in a pair is the start tag, the second tag is the end tag
* The end tag is written like the start tag, but with a slash before the tag name

* HTML태그들은 보통 <p>그리고</p>쌍으로 된 형태다.
* 첫번째 태그의 쌍은 시작 태그이고, 두번째 태그는 끝 태그이다.
* 종료 태그는 시작 태그와 동일하게 쓴다, 하지만 태그이름 앞에 슬래쉬를 함께 쓴다. 

The start tag is often called the opening tag. The end tag is often called the closing tag.

시작 태그는 자주 여는 태그라고도 불린다. 끝 태그는 닫는 태그 라고불린다.

### Web Browsers

### 웹 브라우저

The purpose of a web browser (Chrome, IE, Firefox, Safari) is to read HTML documents and display them.

The browser does not display the HTML tags, but uses them to determine how to display the document:

웹 브라우저의 목적은(크롬, IE, 파이어폭스, 사파리) HTML문서를 판독해서 보여준다.

브라우저에 HTML태그를 표시는 하지만 문서에 보여지기위한 방법을 결정하지 않는다:

### HTML Page Structure

### HTML 페이지 구조

Below is a visualization of an HTML page structure:

아래 보인는 것은 HTML페이지의 구조이다.

<pre class="prettyprint">
&lt;html&gt;
	&lt;head&gt;
		&lt;title&gt; Page title &lt;/title&gt;
	&lt;head&gt;

	&lt;body&gt;
		&lt;h1&gt; This is a heading &lt;/h1&gt;
		&lt;p&gt;This is a paragraph &lt;/p&gt;
		&lt;p&gt; This is another paragraph &lt;p&gt;
	&lt;/body&gt;
&lt;/html&gt;
</pre>
Only the <body> area (the white area) is displayed by the browser.

무조건 <body>지역 (흰색지역)만 화면에 보여진다.

### The <!DOCTYPE> Declaration

### <!DOCTYPE>선언문

The <!DOCTYPE> declaration helps the browser to display a web page correctly.

There are different document types on the web.

To display a document correctly, the browser must know both type and version.

The doctype declaration is not case sensitive. All cases are acceptable:

정확히 말해서 <!DOCUTYPE>은 브라우저에 보이기위에 웹페이지에 선언한다.

웹에는 다른 문서 종류가 있다.

정확하게 된 문서를 표시할려면, 브라우저의 버전과 종류 두가지다 알아야한다.

DOCUTYPE은 대수문자를 구분하지 않는다. 모든 경우에 해당된다.


<pre class="prettyprint">
&lt;!DOCTYPE html&gt;

&lt;!DOCTYPE HTML&gt;

&lt;!doctype html&gt;

&lt;!Doctype Html&gt;
</pre>

## HTML Editors

## HTML 편집

### Write HTML Using Notepad or TextEdit

### HTML은 메모장 또는 텍스트편집기를 사용한다.

HTML can be edited by using professional HTML editors like:

  * Microsoft WebMatrix
  * Sublime Text

However, for learning HTML we recommend a text editor like Notepad (PC) or TextEdit (Mac).

We believe using a simple text editor is a good way to learn HTML.

Follow the 4 steps below to create your first web page with Notepad.

HTML은 전문가가 편집하기위해 사용하는 HTML편집기:

  * 마이크로소프트 웹메트릭스
  * 서브라임 텍스트

우리는 메모장이나(pc) 텍스트편집기(Mac)로 배우는 것을 추천한다.

우리는 간략하게 문자 편집기를 사용해을 좋은 HTML을 배우는 방법이다.

---

### Step 1: Open Notepad

### 1단계 : 메모장 열기

To open Notepad in Windows 7 or earlier:

Click Start (bottom left on your screen). Click All Programs. Click Accessories. Click Notepad.

To open Notepad in Windows 8 or later:

Open the Start Screen (the window symbol at the bottom left on your screen). Type Notepad.

일단 윈도우 7에 있는 메모장을 연다:

시작을 눌러서(당신의 화면 왼쪽밑에). 프로그램을 전부 누른다. 악세서리 누른다. 메모장 누른다.

다음은 윈도우 8의 메모장을 연다:

시작 화면에(당신의 화면 아래에 윈도우 표시).메모장을 시작한다.


---

### Step 2: Write Some HTML

### 2단계 : 일부의 HTML을 쓴다

Write or copy some HTML into Notepad.

메모장에 일부의 HTML을 복사해서 쓴다.

<pre class="prettyprint">
&lt;!DOCTYPE html&gt;
&lt;html&gt;
&lt;body&gt;

&lt;h1&gt; My First Heading &lt;/h1&gt;
		
&lt;p&gt; My first paragraph. &lt;p&gt;

&lt;/body&gt;
&lt;/html&gt;
</pre>

![](http://www.w3schools.com/html/img_notepad.png)

---

### Step 3: Save the HTML Page

### 3단계 : HTML 페이지 저장

Save the file on your computer.

Select File > Save as in the Notepad menu.

Name the file "index.html" or any other name ending with html or htm.

UTF-8 is the preferred encoding for HTML files.

ANSI encoding covers US and Western European characters only.

당신의 컴퓨터에 파일을 저장한다.

메모장의 메뉴에서 파일 > 저장 을 선택한다.

파일의 이름은 "index.html"로한다 다른 이름 끝에는  html이나 htm이라고한다.

UTF-8은 HTML 파일에서 선호하는 코드화이다

ANSI 코드화는 미국과 유럽에서 쓴다.

![](http://www.w3schools.com/html/img_saveas.png)

You can use either .htm or .html as file extension. There is no difference, it is up to you.

당신은 확장자로 htm이나 html파일 어느쪽이든 사용한다. 차이는 크지 않다.  

---

### Step 4: View HTML Page in Your Browser

### 4단계 : HTML페이지를 당신의 브라우저에서 보기

Open the saved HTML file in your favorite browser. The result will look much like this:

당신이 브라우저에서 저장한 HTML파일을 열어 그것의 결과는 보이는 이것과 같다:

![](http://www.w3schools.com/html/img_chrome.png)

To open a file in a browser, double click on the file, or right-click, and choose open with.

브라우저에서 파일을열려면 오른쪽 2번클릭이나 오른쪽 클릭하고 열기 를 선택한다.
